＃專案簡介
本專案為復刻 nerflix 前台畫面及串接 TMDB 取得電影資訊及 stripe 金流實現訂閱制功能，資料庫則是使用 firebase

> 網站連結 https://react-netflix-a0aad.web.app/

## 使用技術
* 前端框架
  * React
* 第三方套件
  * Redux Toolkit
  * React Router
  * styled-components
* 第三方 API
  * stipe
  * TMDB
